# 3D TRONICA Manager 🖨️

Aplicación web para gestionar el emprendimiento de impresión 3D. Sin instalación, sin dependencias — funciona directamente en el navegador.

## 🚀 Ver en vivo

Una vez subido a GitHub, activá **GitHub Pages** en Settings → Pages → Deploy from branch `main` / `/(root)`.

## 📋 Secciones

| Sección | Descripción |
|---|---|
| **Pedidos** | Registrá órdenes con cliente, producto, material, precio y estado |
| **Inventario** | Controlá filamentos, resinas y consumibles con alertas de stock bajo |
| **Gastos** | Llevá el registro de todos los egresos por categoría y mes |
| **Presupuestos** | Creá cotizaciones y convertílas en pedidos con un clic |

## 💾 Almacenamiento

Los datos se guardan en el `localStorage` del navegador. Podés exportar e importar backups en formato `.json` con los botones de la barra superior.
